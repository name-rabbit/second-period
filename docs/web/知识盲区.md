#### 深思钩子函数的与回调函数的区别

1.钩子函数与回调函数都是事件处理函数

 2.钩子函数是指windows的消息处理机制下，**捕获消息的时候**立即执行

 3.回调函数并不能参与消息处理的过程，所以，它是在**消息捕获结束后**才执行的函数

钩子函数

```js
$('button').click(()=>{
console.log('我被点击了!')
})
```

回调函数

```js
$("p").on("click", function(){
alert( $(this).text() );
});
```

my:回调就是函数里面调函数
